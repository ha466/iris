# Iris AI - Your Intelligent Companion

Welcome to the Iris AI project! This repository contains the code for an interactive website introducing Iris, an AI powered by Llama 3.3, Whisper-large-v3, and Deep Speech TTS.

## Features

- Modern, responsive design
- Animated UI elements using Framer Motion
- Integration with GitHub for easy deployment and collaboration

## Getting Started

To run this project locally:

1. Clone the repository:
   \`\`\`
   git clone https://github.com/your-username/iris-ai-intro.git
   cd iris-ai-intro
   \`\`\`

2. Install dependencies:
   \`\`\`
   npm install
   \`\`\`

3. Run the development server:
   \`\`\`
   npm run dev
   \`\`\`

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

## Technologies Used

- Next.js
- React
- Framer Motion
- Tailwind CSS
